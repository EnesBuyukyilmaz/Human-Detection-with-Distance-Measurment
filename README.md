# Human-Detection-with-Distance-Measurment

  In this project, i aimed to find distance of a human that is detected by the yolov3 model who is closer than 100 cm . If all condtions are provided then led will turn on.  

All other information will be displayed on I2C 2x16 LCD display .

## Necessary Equipments for this project 
  - Jetson Nano (will be working as processing unit)
  - HCS404 Distance Sensor ( will calculate the distance of a human)
  - Led
  - LCD Display
